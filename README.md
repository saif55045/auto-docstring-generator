## 📦 Getting Started

### Prerequisites
- Python 3.9+
- Jupyter Notebook or JupyterLab

### Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/saif55045/code-docstring.git
   cd code-docstring
   ```

2. **Install dependencies and launch:**
   ```bash
   pip install jupyter
   jupyter notebook code-docstring.ipynb
   ```
